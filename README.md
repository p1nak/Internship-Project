# Internship Projects Portfolio

Welcome to my internship projects repository! This repository contains a collection of three distinct industrial software solutions developed during my internship. Each project tackles a specific challenge in manufacturing, quality control, and industrial IoT (IIoT).

## 📁 Projects Overview

### 1. 🏭 Sand Plant Energy & Telemetry Monitoring (`PLC Project`)
A live Industrial IoT (IIoT) monitoring system simulation for a sand mixing/muller plant. 
- **Core Functionality**: Generates realistic machine telemetry data, logs it locally, and streams it live to InfluxDB Cloud for real-time visualization in Grafana.
- **Tech Stack**: Python, Pandas, InfluxDB, Grafana.
- **Key Features**: Realistic machine telemetry simulation, local CSV logging, and cloud-based live dashboarding.
- **Documentation**: [Read the full guide](./PLC%20Project/README.md)

### 2. 📋 Digital Quality Control System (`QC control`)
A comprehensive, web-based Quality Control (QC) management system designed to streamline the inspection of Electrical Control Panels.
- **Core Functionality**: Digitizes the QC process with role-based access, dynamic dashboards, digital inspection forms, and defect photo tracking.
- **Tech Stack**: Python (Flask), SQLite3, HTML/CSS/JS, Bootstrap 5.
- **Key Features**: Secure role-based login, interactive data dashboard, premium UI/UX (Glassmorphism, Dark/Light modes), and print-optimized reporting.
- **Documentation**: [Read the full guide](./QC%20control/README.md)

### 3. 🔲 QR Machine Documentation System (`QR Generator`)
A robust web application designed to generate and manage QR codes for machine documentation, tailored for industrial shop floor usage.
- **Core Functionality**: Allows engineers and admins to generate trackable QR codes linked to essential machine documentation and setup guides.
- **Tech Stack**: Python (Flask).
- **Key Features**: Streamlined setup process, role-based access control (Admin/Engineer accounts), and a user-friendly web interface for shop floor operators.
- **Documentation**: [Read the full guide](./QR%20Generator/README.md)
